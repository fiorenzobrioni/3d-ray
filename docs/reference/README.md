# 3D-Ray Scene Reference / Riferimento Scene di 3D-Ray

---

## English

Complete technical reference for every YAML key accepted by the 3D-Ray engine.
Covers world settings, camera parameters, all material types, every geometric primitive,
light sources, procedural and image textures, CSG, groups, templates, and the import system.

**[→ Scene Reference (English)](./scene-reference.md)**
