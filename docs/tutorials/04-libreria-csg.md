# Libreria dei Preset CSG (Constructive Solid Geometry)

Benvenuto nel catalogo delle formeBooleane complesse. La CSG permette di scolpire geometrie personalizzate sottraendo, intersecando o unendo primitive base. In questa libreria troverai snippet pronti per lenti ottiche, parti meccaniche, complessi alberi di operazioni booleane e scene starter pre-bilanciate per il rendering CSG.
